خانه (HOME)

به «آموزش MySQL» خوش آمدی! اینجا ساده پیش می رویم. MySQL یک «سامانه مدیریت پایگاه داده رابطه ای (RDBMS)» است. «پایگاه داده (Database)» یعنی جایی برای نگهداری داده های مرتب. «رابطه ای (Relational)» یعنی جدول ها با هم ارتباط دارند. مثل دفتر نمره های کلاس های مختلف.

شروع سریع آموزش MySQL

MySQL رایگان و متن باز است. برای برنامه های کوچک و بزرگ مناسب است. «RDBMS» یعنی موتور اجرای پرس وجوهای جدولی. مثل اکسل، اما حرفه ای و سریع. با دستورها، داده ها را می خوانی، می نویسی، ویرایش می کنی.

اولین پرس وجو (Query) ساده

«پرس وجو (Query)» یعنی درخواست از پایگاه داده. این مثال همه ردیف های جدول «Customers» را نشان می دهد. شبیه نگاه کردن به فهرست همه دانش آموزان.

SELECT *
FROM Customers;

مشاهده در ادیتور

نکته: ستاره (*) یعنی «همه ستون ها». اگر ستون خاص می خواهی، نامش را بنویس. مثلاً SELECT Name.

تمرین ها و مثال ها

بعد از هر فصل، تمرین داری. تمرین یعنی حل مسئله کوتاه. با تمرین، یادگیری عمیق تر می شود. همچنین مثال های زیاد می بینی. مثال یعنی الگو. با الگو سریع تر می سازی.

همه تمرین های MySQL را بعد از مطالعه انجام بده.

برای نمونه های بیشتر، بخش مثال های MySQL را ببین. آنجا الگوهای آماده داری.

آزمون و سنجش

با «آزمون MySQL» خودت را بسنج. آزمون مثل مسابقه کوچک است. نتیجه بهت می گوید کجا قوی تر شوی. سپس دوباره درس را مرور کن.

برای شروع مسیر، از مقدمه MySQL برو. این صفحه مقدمه، کل مسیر «آموزش MySQL» را روشن می کند.

جمع بندی سریع

  • MySQL رایگان و متن باز است.
  • Query یعنی درخواست از داده ها.
  • SELECT برای خواندن داده هاست.
  • تمرین و مثال، یادگیری را محکم می کند.
  • آزمون، پیشرفتت را نشان می دهد.
