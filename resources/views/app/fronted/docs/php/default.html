خانه (HOME)

اینجا با آموزش PHP شروع می کنیم. PHP یک «زبان اسکریپت نویسی (Scripting Language)» سمت سرور است. یعنی روی سرور می دود و صفحه پویا می سازد. مثل وقتی نمره ها را در سایت مدرسه می بینی.

PHP چیه؟ (PHP)

PHP ابزار ساخت وب پویاست. «سمت سرور» یعنی پردازش در سرور انجام می شود. سپس نتیجه به مرورگر می رسد. مثل سفارش غذا که در آشپزخانه آماده می شود.

اولین اسکریپت PHP

این کد یک متن ساده چاپ می کند. «echo» یعنی خروجی گرفتن. فایل را در سرور اجرا کن تا نتیجه را ببینی.

<!DOCTYPE html>
<html>
<body>
  <?php
  echo "My first PHP script!";
  ?>
</body>
</html>
