خانه (HOME)

اینجا شروع ساده آموزش CSS است. CSS یعنی «شیوه نامه های آبشاری (Cascading Style Sheets)». این زبان ظاهر HTML را می سازد. مثل انتخاب لباس برای کاراکتر بازی است؛ متن همان کاراکتر است، CSS لباسی خوش رنگ می پوشاند.

CSS چیست؟

CSS سبک صفحه را تعیین می کند. سبک یعنی رنگ، فاصله، اندازه و چیدمان. بنابراین یک فایل CSS می تواند ظاهر همه صفحات سایت را عوض کند.

مثال سریع از آموزش CSS

در این مثال، پس زمینه آبی می شود، عنوان وسط قرار می گیرد، و متن خواناتر می شود.

<style>
body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}
</style>
<h1>سلام CSS</h1>
<p>این یک پاراگراف نمونه است.</p>

مشاهده در ادیتور

چرا از CSS استفاده کنیم؟

چون یک تغییر کوچک، کل سایت را بهتر می کند. همچنین برای موبایل و دسکتاپ می توان نمایش متفاوت داشت.

سه گام شروع سریع

  1. یک فایل style.css بساز.
  2. کدهای ساده آموزش CSS را وارد کن.
  3. فایل را به صفحه HTML لینک کن.

نکته: اگر تازه کاری، قدم به قدم جلو برو. هر بار یک ویژگی را تست کن.

مثال ها، تمرین ها و آزمون

در منبع اصلی، برای هر فصل مثال و محیط ویرایش آنلاین هست. با تمرین ها خودت را محک بزن. سپس با آزمون، نتیجه را ببین.

مسیر پیشنهادی مطالعه

به ترتیب فصل ها پیش برو. از ساده به پیشرفته حرکت کن. برای مقدمه، صفحه آموزش CSS را ببین. برای سازگاری موبایل، طراحی واکنش گرا را مرور کن.

جمع بندی سریع

  • CSS ظاهر HTML را می سازد.
  • یک فایل CSS، کل سایت را تغییر می دهد.
  • با مثال ها سریع تر یاد می گیری.
  • تمرین و آزمون، یادگیری را محکم می کند.
