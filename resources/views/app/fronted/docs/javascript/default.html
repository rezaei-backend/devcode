خانه (HOME)

به «آموزش جاوااسکریپت» خوش آمدی! جاوااسکریپت (JavaScript) زبان برنامه نویسی وب است. با آن صفحه ها زنده می شوند. این دوره ساده است. قدم به قدم می رویم. آهسته بخوان. سپس تمرین کن.

شروع سریع با آموزش جاوااسکریپت

اینجا یک مثال ساده می بینیم. «رویداد (Event)» یعنی اتفاقی مثل کلیک. با کلیک، متن جایگزین می شود.

function showNow() {
  var now = Date();
  var el = document.getElementById('demo');
  el.innerHTML = now;
}

مشاهده در ادیتور

چرا جاوااسکریپت مهم است؟

سه رفیق وب داریم: HTML برای محتوا، CSS برای ظاهر، و JavaScript برای رفتار. با این سه تا، سایت کامل می شود. مثالش مثل کلاس، متن، و حرکت انیمیشن است.

چگونه از منو استفاده کنم؟

از بالا تا پایین پیش برو. ترتیب مهم است. اگر نمایشگر بزرگ داری، منو سمت چپ می ماند. روی موبایل، آیکون منو را بزن.

با مثال ها یاد بگیر

مثال از هزار کلمه بهتر است. هر توضیح، یک مثال دارد. روی «Try it» بزن و کد را تغییر بده. اشتباه کنی، بهتر یاد می گیری.

سرعت یادگیری دست خودت است

اگر سخت شد، کمی استراحت کن. بعد دوباره بخوان. حتماً تمام مثال ها را اجرا کن. تمرین زیاد، تو را برنامه نویس زرنگ می کند.

سؤال های معمول

جاوااسکریپت را نصب نکن. مرورگر تو دارد. رایگان است. همین حالا می توانی یاد بگیری و تمرین کنی.

تمرین، آزمون، و مرجع

پس از هر فصل تمرین هست. یک بخش تمرین هم خواهیم داشت. برای جمع بندی، آزمون کوتاه عالی است. نیاز به جزئیات داشتی، مرجع جاوااسکریپت را ببین.

جمع بندی سریع

  • جاوااسکریپت رفتار صفحه را می سازد.
  • با مثال ها جلو برو.
  • هر روز کمی تمرین کن.
  • سؤال داشتی، مرجع را ببین.
